#  Caesar Cipher Cracker (Brute Force)

##  Описание
Это консольная утилита для стопроцентного взлома классического шифра Цезаря. В отличие от частотного анализатора, который может ошибаться на коротких сообщениях, этот скрипт использует метод **Brute Force (Полный перебор)**. Он мгновенно генерирует и выводит на экран все возможные варианты сдвига английского алфавита.

##  Особенности
* **Абсолютная надежность:** Перебирает все 25 возможных вариантов математического ключа за долю секунды.
* **Игнорирование ловушек:** Легко справляется с одиночными словами и текстами, в которых намеренно спрятаны частые буквы.
* **Чистый вывод:** Благодаря методу `.isalpha()` скрипт сдвигает только буквы, оставляя пробелы, цифры и знаки препинания на своих законных местах.
* **Поиск ключа:** Рядом с каждой строкой выводится номер использованного сдвига, что позволяет сразу узнать секретный ключ.

##  Запуск и использование
1. Склонируйте репозиторий или скачайте файл скрипта.
2. Запустите скрипт через терминал (консоль):
   ```bash
   python caesar_cipher_brute_force.py
   ```
3. Вставьте зашифрованный текст (на английском языке) при появлении запроса и нажмите Enter.
4. Просмотрите список из 25 вариантов на экране. Тот вариант, который читается как осмысленный текст — и есть правильный ответ!
